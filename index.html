Download for Windows

MSI installer · auto-updates after install · all releases

Three reasons to install

Live, every 60 s

The hidden webview reads your session and pings /usage on a steady tick — no console paste, no extra tabs.

Tray-resident

Park it on a second monitor or hide it in the tray. Toggle "Start with Windows" once and forget about it.

Updates itself

Install once. New versions are signed (Ed25519), auto-downloaded from GitHub, and applied on the next launch.

Three-step setup

  1. Download & run the MSI. Windows SmartScreen will warn — click More info → Run anyway. The installer drops a tray icon and opens the main window.
  2. Sign in to claude.ai once. An embedded webview pops up on first launch. Use email login (Google OAuth is blocked inside embedded webviews — that's a Google policy, not ours).
  3. That's it. The widget renders your live quota and refreshes every minute. Toggle Start with Windows in the gear menu if you want it always-on.

Or use the web mini version

Web mini version (no install) DevTools paste · works anywhere
